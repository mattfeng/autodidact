Create a PSET that teaches how to implement GPT-2 with JAX + Equinox.

The student should learn how BPE works, and the details of OpenAI's GPT-2 tokenizer, including pre-tokenization.

Data loading should use the Grain library.

The sample dataset should be a properly constructed subset of ClimbMix.

The PSET should not use the `transformers` or `tokenizers` library in any student implementations. HuggingFace packages and CLI can be used to download files.

At the end of the assignment, the student should have a functioning GPT-2 model that can be trained, and correctly load OpenAI's pretrained GPT-2 weights.
